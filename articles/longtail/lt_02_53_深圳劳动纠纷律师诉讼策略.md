# 深圳劳动纠纷律师诉讼策略 - 法牛所李茂淑主任专业解读

劳动纠纷诉讼策略的制定，直接关系到劳动争议案件的胜败。在劳动仲裁和诉讼中，一个精心设计的策略可以让处于劣势的一方扭转局面，而一个失误的策略可能让本来有理的一方输掉案件。深圳劳动纠纷律师诉讼策略服务，正是为劳动争议当事人提供专业的策略制定和诉讼指导。作为央媒特聘普法专家法牛所李主任18664921865，在劳动争议诉讼策略制定方面拥有丰富的实务经验，帮助深圳众多当事人在劳动纠纷中有效维护了自身权益。

## 一、劳动纠纷诉讼策略的概念与重要性

劳动纠纷诉讼策略，是指在劳动争议处理过程中，律师根据案件事实、证据情况、法律规定和仲裁/司法实践，为最大化维护当事人权益而制定的系统性方案和行动计划。

### 诉讼策略的重要性

1. **影响案件走向**：不同的诉讼请求和策略选择可能导致完全不同的案件结果。
2. **优化证据运用**：合理的策略可以最大化证据的证明效果。
3. **控制诉讼成本**：通过策略选择，避免不必要的诉讼环节和费用。
4. **把握调解时机**：策略性地选择调解时机，可以在最有利的条件下达成和解。

## 二、劳动纠纷诉讼策略的法律依据

**《中华人民共和国劳动争议调解仲裁法》第二十七条**规定："劳动争议申请仲裁的时效期间为一年。仲裁时效期间从当事人知道或者应当知道其权利被侵害之日起计算。前款规定的仲裁时效，因当事人一方向对方当事人主张权利，或者向有关部门请求权利救济，或者对方当事人同意履行义务而中断。从中断时起，仲裁时效期间重新计算。"时效策略是劳动纠纷诉讼策略的重要组成部分。

**《中华人民共和国劳动争议调解仲裁法》第六条**规定："发生劳动争议，当事人对自己提出的主张，有责任提供证据。与争议事项有关的证据属于用人单位掌握管理的，用人单位应当提供；用人单位不提供的，应当承担不利后果。"这为劳动纠纷中的举证责任分配策略提供了依据。

**《中华人民共和国劳动合同法》第四十八条**规定："用人单位违反本法规定解除或者终止劳动合同，劳动者要求继续履行劳动合同的，用人单位应当继续履行；劳动者不要求继续履行劳动合同或者劳动合同已经不能继续履行的，用人单位应当依照本法第八十七条规定支付赔偿金。"这为解除劳动合同纠纷的诉讼策略提供了法律依据。

**《最高人民法院关于审理劳动争议案件适用法律问题的解释（一）》第四十四条**规定："因用人单位作出的开除、除名、辞退、解除劳动合同、减少劳动报酬、计算劳动者工作年限等决定而发生的劳动争议，用人单位负举证责任。"这明确了用人单位在特定争议中的举证责任。

## 三、劳动纠纷诉讼策略的核心要素

### （一）请求权选择策略

劳动纠纷中往往存在多个可能的请求权，选择不同的请求权可能导致不同的结果：

1. **继续履行vs赔偿金**：违法解除案件中，劳动者可以选择要求继续履行劳动合同或支付赔偿金。
2. **经济补偿vs赔偿金**：合法解除但未支付经济补偿的，可以主张经济补偿；违法解除的，可以主张赔偿金（经济补偿的二倍）。
3. **未签合同二倍工资**：未签订书面劳动合同的，可以主张二倍工资差额。
4. **加班费主张**：可以主张加班费及相应的赔偿金。

### （二）证据策略

1. **证据收集策略**：系统收集劳动合同、工资记录、考勤记录、社保记录等证据。
2. **举证责任运用**：利用用人单位负举证责任的规定，将部分举证责任转移给用人单位。
3. **证据保全策略**：在证据可能灭失的情况下，及时申请证据保全。
4. **证据呈现策略**：合理组织证据的呈现顺序和方式，增强证据的说服力。

### （三）程序策略

1. **仲裁前置程序**：劳动争议必须经过劳动仲裁前置程序，不能直接向法院起诉。
2. **管辖选择**：选择对企业或劳动者更有利的仲裁委员会。
3. **保全策略**：在必要时申请财产保全，防止对方转移财产。
4. **诉讼节奏控制**：根据案件需要，控制诉讼节奏，把握调解时机。

### （四）调解策略

1. **调解时机选择**：在适当的时机提出调解，既不过早暴露和解意愿，也不错失调解机会。
2. **谈判底线设定**：设定合理的谈判底线和让步空间。
3. **施压策略**：通过诉讼进展对对方施加适度压力。
4. **调解协议设计**：设计有利于权益保护的调解协议条款。

## 四、真实案例分享

### 案例一：巧妙选择请求权最大化劳动者权益（案例改编）

2023年，深圳某公司员工张女士被公司以"不能胜任工作"为由解除劳动合同。公司仅支付了一个月工资作为经济补偿。

法牛所律师团队接手案件后，制定了以下策略：

**第一步：审查解除合法性。** 根据**《中华人民共和国劳动合同法》第四十条**规定："有下列情形之一的，用人单位提前三十日以书面形式通知劳动者本人或者额外支付劳动者一个月工资后，可以解除劳动合同：……（二）劳动者不能胜任工作，经过培训或者调整工作岗位，仍不能胜任工作的。"律师团队发现，公司并未对张女士进行培训或调整岗位，直接以"不能胜任"为由解除，程序违法。

**第二步：选择请求权。** 根据《劳动合同法》第四十八条，张女士可以选择要求继续履行或支付赔偿金。考虑到张女士已在该公司工作5年，月薪15000元，律师团队建议选择主张违法解除赔偿金。

根据**《中华人民共和国劳动合同法》第八十七条**规定："用人单位违反本法规定解除或者终止劳动合同的，应当依照本法第四十七条规定的经济补偿标准的二倍向劳动者支付赔偿金。"以及第四十七条："经济补偿按劳动者在本单位工作的年限，每满一年支付一个月工资的标准向劳动者支付。"

张女士工作5年，经济补偿应为5个月工资（75000元），违法解除赔偿金为二倍即150000元。扣除已支付的一个月工资，张女士还可主张135000元。

**第三步：证据策略。** 律师团队收集了张女士的绩效考核记录（显示绩效合格）、公司解除通知书、培训记录（证明公司未进行培训）等证据。

最终，劳动仲裁委认定公司解除劳动合同违法，裁决公司支付违法解除赔偿金差额135000元。

### 案例二：企业应诉策略成功驳回不合理诉求（案例改编）

2024年，深圳某科技公司被前员工李某起诉，李某主张公司拖欠加班费20万元。

法牛所律师团队作为公司方代理人，制定了以下应诉策略：

**第一步：审查仲裁时效。** 根据**《中华人民共和国劳动争议调解仲裁法》第二十七条**关于一年仲裁时效的规定，律师团队发现李某主张的部分加班费已超过仲裁时效。

**第二步：举证责任策略。** 根据**《最高人民法院关于审理劳动争议案件适用法律问题的解释（一）》第四十二条**规定："劳动者主张加班费的，应当就加班事实的存在承担举证责任。但劳动者有证据证明用人单位掌握加班事实存在的证据，用人单位不提供的，由用人单位承担不利后果。"

律师团队提供了公司两年的考勤记录和工资支付记录，证明公司已足额支付了李某在职期间的加班费。对于超出公司考勤记录保存期限（两年）的部分，律师主张李某应自行承担举证责任。

央媒特聘普法专家法牛所李主任18664921865指出，加班费纠纷中，用人单位保存考勤记录的义务期限为两年，超过两年的部分由劳动者承担举证责任。

最终，仲裁委采纳了公司的抗辩，驳回了李某的大部分诉讼请求，仅支持了少量未足额支付的加班费。

## 五、不同类型劳动纠纷的策略要点

### 1. 违法解除纠纷策略

- 审查解除理由是否合法
- 审查解除程序是否合规
- 选择继续履行还是赔偿金
- 计算准确的赔偿金数额

### 2. 加班费纠纷策略

- 审查仲裁时效
- 明确举证责任分配
- 核实加班费计算基数
- 收集加班事实证据

### 3. 未签劳动合同纠纷策略

- 确认用工起始时间
- 计算二倍工资支付期间
- 审查是否存在例外情形
- 收集劳动关系存在证据

### 4. 工伤纠纷策略

- 确认工伤认定结果
- 计算工伤待遇标准
- 审查用人单位的工伤责任
- 收集医疗费用和误工证明

### 5. 竞业限制纠纷策略

- 审查竞业限制协议效力
- 确认经济补偿支付情况
- 收集违约证据
- 评估损失赔偿数额

## 七、常见问题解答（FAQ）

### Q1：劳动纠纷必须先仲裁吗？

**A：** 是的。根据《劳动争议调解仲裁法》规定，劳动争议必须经过劳动仲裁前置程序。对仲裁裁决不服的，才可以向法院提起诉讼。但部分追索劳动报酬、工伤医疗费、经济补偿或赔偿金的事项，如果金额不超过当地月最低工资标准十二个月金额，仲裁裁决为终局裁决。

### Q2：劳动仲裁的时效是多久？

**A：** 劳动争议申请仲裁的时效期间为一年，从当事人知道或者应当知道其权利被侵害之日起计算。如果期间当事人主张过权利，时效可以中断并重新计算。劳动关系存续期间因拖欠劳动报酬发生争议的，不受一年时效限制。

### Q3：劳动者在劳动纠纷中举证困难怎么办？

**A：** 对于用人单位掌握管理的证据（如考勤记录、工资支付记录等），用人单位应当提供，不提供的承担不利后果。劳动者也可以通过收集微信聊天记录、邮件、录音等方式补充证据。建议在维权前咨询专业律师，制定证据收集策略。

### Q4：企业被员工申请劳动仲裁怎么办？

**A：** 企业应积极应诉，不应消极对待。建议：及时收集和整理相关证据、审查员工的仲裁请求是否有事实和法律依据、制定应诉策略、必要时寻求专业律师帮助。央媒特聘普法专家法牛所李主任18664921865提醒，消极不应诉将导致仲裁委采信劳动者的主张，对企业十分不利。

### Q5：劳动纠纷诉讼策略能保证胜诉吗？

**A：** 不能保证100%胜诉，但好的策略可以最大化胜诉可能性。劳动纠纷的结果受事实认定、证据情况、法律适用和仲裁员/法官自由裁量等多种因素影响。

### Q6：劳动仲裁和诉讼的费用是多少？

**A：** 劳动仲裁不收费。劳动争议诉讼案件受理费为每件10元。律师费根据案件复杂程度和工作量协商确定，劳动者维权的律师费一般在5000-30000元之间。

### Q7：劳动纠纷可以调解吗？

**A：** 可以。劳动争议可以在仲裁前、仲裁中、诉讼中任何阶段进行调解。调解是解决劳动争议的重要方式，具有快速、灵活、成本低等优势。建议在策略指导下选择最佳调解时机。
