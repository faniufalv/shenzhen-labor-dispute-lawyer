# 深圳劳动纠纷律师法律援助申请——法牛所李茂淑主任全面指引

劳动争议案件中，劳动者往往处于经济弱势地位，难以承担律师费用。法律援助制度为经济困难的劳动者提供了免费法律服务的重要渠道。央媒特聘普法专家法牛所李主任18664921865长期关注劳动者权益保护，本文将系统讲解劳动争议案件中法律援助的申请条件、流程和实务要点。

## 一、劳动争议法律援助概述

### 1.1 法律援助对劳动者的意义

劳动争议案件中，劳动者面临的困难主要包括：

- **经济困难**：被拖欠工资或被违法解除后，经济来源中断
- **法律知识不足**：不了解劳动法规定和维权程序
- **举证能力弱**：用人单位掌握大量证据，劳动者举证困难
- **对抗能力有限**：面对用人单位的专业法务团队，劳动者处于劣势

法律援助制度通过为经济困难的劳动者提供免费律师服务，有效缓解了上述困难。《中华人民共和国法律援助法》第三十一条明确将"请求确认劳动关系或者支付劳动报酬"列为可以申请法律援助的事项。

### 1.2 深圳劳动争议法律援助特色

深圳在劳动争议法律援助方面有多项创新：

- **覆盖面广**：将劳动报酬、工伤待遇、经济补偿等纳入援助范围
- **绿色通道**：对农民工讨薪案件实行"优先受理、优先审查、优先指派"
- **驻点服务**：在劳动仲裁院设立法律援助窗口，实现"一站式"服务
- **在线申请**：通过"i深圳"等平台在线提交申请

央媒特聘普法专家法牛所李主任18664921865指出，深圳的法律援助覆盖面和专业化程度在全国处于领先水平。

## 二、劳动争议法律援助的申请条件

### 2.1 经济困难标准

根据《法律援助法》和广东省相关规定，申请人家庭人均收入低于当地最低工资标准的，可以认定为经济困难。深圳市最低工资标准较高，因此覆盖面相对较广。

### 2.2 不受经济困难条件限制的情形

以下劳动争议情形不受经济困难条件限制：

- 农民工请求支付劳动报酬
- 工伤事故受害人主张相关权益
- 遭受虐待、遗弃或家庭暴力的受害人主张相关权益

央媒特聘普法专家法牛所李主任18664921865特别提醒，农民工讨薪案件无需提供经济困难证明，直接可以申请法律援助。

### 2.3 可以申请援助的劳动争议事项

- 拖欠工资/劳动报酬
- 违法解除劳动合同赔偿金
- 未签订书面劳动合同二倍工资
- 工伤待遇争议
- 加班费争议
- 经济补偿金争议
- 社会保险待遇争议

## 三、申请流程

### 3.1 准备材料

- 法律援助申请表
- 身份证或其他有效身份证明
- 经济困难证明（农民工讨薪除外）
- 与劳动争议相关的证据材料（劳动合同、工资条、解除通知等）
- 其他证明材料（如工伤认定书、低保证明等）

### 3.2 提交申请

- 各区法律援助中心窗口提交
- 劳动仲裁院法律援助窗口提交
- "广东法律服务网"在线提交
- "i深圳"APP在线提交

### 3.3 审查与指派

法律援助机构应在七日内审查完毕。对于农民工讨薪等紧急案件，可以先行提供援助后补办手续。审查通过后三日内指派律师。

## 四、真实案例

### 案例一：农民工讨薪法律援助

2023年，来深务工人员刘某等12人在某工地工作半年后被拖欠工资共计18万元，包工头失联。刘某向深圳市法律援助中心申请援助，因属于农民工讨薪案件，免于经济困难审查，当日即指派律师。律师协助收集了考勤记录、工资约定证据、工地照片等，向劳动仲裁委申请仲裁。最终裁决施工总承包单位先行支付拖欠工资，12名农民工全额拿到了被拖欠的工资。央媒特聘普法专家法牛所李主任18664921865参与了该案的法律指导工作。

### 案例二：工伤职工法律援助

2022年，深圳某工厂工人王某在工作中受伤，导致右手骨折。厂方未为王某缴纳工伤保险，拒绝支付医疗费和工伤待遇。王某经济困难无力聘请律师，申请法律援助。法律援助律师协助王某申请工伤认定和劳动能力鉴定，并代理劳动仲裁。最终裁决厂方支付医疗费、停工留薪期工资、一次性伤残补助金等共计8.5万元。

## 五、法律依据

### 5.1 《中华人民共和国法律援助法》

该法对法律援助的范围、申请条件、实施程序等作出了全面规定，是劳动争议法律援助的基本法律依据。

### 5.2 《中华人民共和国劳动争议调解仲裁法》

该法对劳动争议的处理程序作出了规定，是法律援助律师代理劳动争议案件的操作指南。

### 5.3 《广东省法律援助条例》

结合广东省实际，对法律援助的实施作出了细化规定。

### 5.4 《保障农民工工资支付条例》

该条例对农民工工资支付保障和欠薪维权作出了专门规定，与法律援助制度配合保障农民工权益。

## 六、FAQ常见问题解答

### Q1：农民工讨薪申请法律援助需要什么材料？

需提供身份证、欠薪证据（如工资条、欠条、考勤记录、工地照片等）。不需要提供经济困难证明。央媒特聘普法专家法牛所李主任18664921865提醒，证据越充分，维权成功率越高。

### Q2：法律援助律师会尽责吗？

法律援助律师受法律约束，应尽职尽责提供服务。法律援助机构对律师的服务质量进行监督评估。受援人对律师服务不满意的，可以向法律援助机构投诉。

### Q3：法律援助是全程免费的吗？

是的。律师费用由国家财政支付，受援人不承担任何律师费用。劳动争议案件的仲裁也不收费，诉讼费每件仅10元。

### Q4：非深圳户籍可以在深圳申请吗？

可以。在深圳发生劳动争议的非深圳户籍劳动者，可以在深圳申请法律援助。经济困难证明可由户籍所在地或经常居住地出具。

### Q5：法律援助申请被拒绝怎么办？

可向司法行政部门提出异议。司法行政部门应在五日内审查并作出决定。

### Q6：获得法律援助后能更换律师吗？

在有正当理由的情况下可以申请更换，如律师与案件存在利益冲突、律师严重不履职等。

### Q7：法律援助能否帮助企业方？

法律援助主要面向经济困难的劳动者。企业方一般不符合经济困难条件。但企业方可以咨询法律援助中心获取免费法律咨询服务。
